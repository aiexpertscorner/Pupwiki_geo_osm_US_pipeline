# Input files used to design this repo

- `pupwiki_geo_data_sources_checklist.xlsx`: source/data checklist with priorities, local paths and official source URLs.
- `geodatapupwiki.xlsm`: earlier workbook used to generate the dataset manifest.
- `pupwiki-dog-services-map-claude-code-pack.zip`: map/frontend prompt pack integrated under `legacy-packs/`.
- `pupwiki_cloudflare_geo_pseo_pack.zip`: Cloudflare/R2 and pSEO strategy integrated under `legacy-packs/`.
- `Maps pupwiki geo.pdf`: masterplan notes for OSM → GeoJSON fix → Astro/MapLibre integration.
