# Licensing and attribution

OpenStreetMap data is available under the Open Database License. Any PupWiki page or map using OSM-derived data should include visible attribution:

```txt
© OpenStreetMap contributors. Data available under ODbL.
```

Recommended notice:

```txt
This page uses OpenStreetMap-derived place data and local boundary matching. Details such as hours, dog access rules and emergency availability can change. Always contact the place directly before visiting.
```

Do not remove OSM attribution from map pages or static POI pages.
