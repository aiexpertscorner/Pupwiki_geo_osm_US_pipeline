# Quality and SEO strategy

## Page generation thresholds

- State pages: generate when a state has data.
- Category pages: generate for core categories.
- State/category pages: 20+ POIs.
- City pages: 10+ POIs.
- City/category pages: 3+ POIs and average quality >= 40.
- POI detail pages: only later, for high-quality records.

## Quality score inputs

- named POI
- usable address
- city/state resolved by boundaries
- contact info
- website
- opening hours
- confidence of tag match

## Warnings to show or use internally

- missing name
- emergency status inferred
- dog access may change
- county fallback used instead of city/place

## Safe wording

Use:

- “reported dog-friendly”
- “listed as a veterinary clinic”
- “call before visiting”
- “data may be incomplete”

Avoid:

- “verified”
- “best”
- “top-rated”
- “open now”
- “24/7”
